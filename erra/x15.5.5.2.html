(Первый параграф: свойства отдельных символов не должны иметь ограничений "индекса массива")

Объекты String используют разновидность внутреннего метода [[GetOwnProperty]], применяемого и для других родных объектов ECMAScript (см. пункт 8.12.1). Этот специальный внутренний метод используется для добавления доступа для указания индекса массива именованных свойств, соответствующих отдельным символам объектов String.

(Исправления в алгоритме)

  1. Если ToString(abs(ToInteger(P))) не имеет такого же значения, что и P индекс массива (15.4), вернуть undefined.

  2. Пусть str будет строковым значением внутреннего свойства [[PrimitiveValue]] для S.

  3. Пусть index будет ToUint32 ToInteger(P).
