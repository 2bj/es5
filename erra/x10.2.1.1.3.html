Конкретный метод записи окружения SetMutableBinding для декларативных записей окружения предпринимает попытку изменить связанное значение текущей привязки идентификатора, имя которого представляет собой значение аргумента N, на значение аргумента V. Привязка для N уже должна существовать. Если эта привязка является неизменяемой, то всегда будет генерироваться исключение TypeError, если S равно true. Аргумент S игнорируется, поскольку строгий режим не изменяет значение привязок в декларативных записях окружения.

  1. Иначе, это является попыткой изменить значение неизменяемой привязки, поэтому, если S равно true, сгенерировать исключение TypeError.
