  • В остальных случаях, когда в операции не участвуют ни бесконечность, ни ноль, ни NaN, остаток для чисел с плавающей запятой r для делимого n и делителя d определяется математическим отношением r = n − (d * q), где q – целое число, которое является отрицательным, только если n/d отрицательно, и положительным, только если n/d положительно, и чьё значение по модулю является наибольшим значением, не превышающим значения по модулю истинного математического отношения чисел n и d. Значение r вычисляется и округляется до ближайшего представимого значения с использованием режима "округления до ближайшего значения" в соответствии со стандартом IEEE 754.
