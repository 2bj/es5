Различия в web-платформах

Для String.prototype имеется ещё несколько дополнительных методов для завёрстывания текста в элементах HTML (все эти методы являются родовыми: не требуется, чтобы значение this было объектом типа String):

Алгоритм ToHTMLTag(tag_name, content, attribute_name, attribute_value):

  1. если attribute_name равно undefined, вернуть "<" + tag_name + ">" + content + "</" + tag_name + ">"
  2. иначе вернуть "<" + tag_name + " " + attribute_name + "=\"" + attribute_value + "\">" + content + "</" + tag_name + ">"

Также см. http://wiki.whatwg.org/wiki/Web_ECMAScript#15.5.4_Properties_of_the_String_Prototype_Object

String.prototype.anchor(name)
  1. Пусть content будет ToString(this)
  2. Пусть attribute_value будет ToString(name)
  3. Вернуть ToHTMLTag("a", content, "name", attribute_value)
